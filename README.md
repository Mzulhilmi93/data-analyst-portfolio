## Project 1 - E-Commerce Sales Analysis

**Tools:** Python, pandas, matplotlib

**Dataset:** E-Commerce Sales Dataset (100 orders)

### Key Insight
- Total Revenue: $512,383.00
- Electronics was the top category at $258,523.00 (50.5%)
- Laptop was the best performing product at $162,098.00
- Electronics and Accessories were nearly equal in revenue - suggesting balanced inventory is working well
